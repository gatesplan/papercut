# MinerU 요구 사양 조사 (2026-07-09)

## 확인된 사항

- MinerU는 로컬에서 딥러닝 모델을 직접 구동하는 도구 (레이아웃 분석, OCR, 수식/표 인식 등). API 서비스 아님.
- 백엔드별 최소 VRAM [근거, https://opendatalab.github.io/MinerU/quick_start/]
  - pipeline: 최소 4GB. CPU-only도 가능(느림). 단계 간 VRAM 회수 로직 있음.
  - vlm 계열(vlm-transformers, vlm-vllm 등): 최소 8GB. Turing 이상 아키텍처.
- vllm 서버(mineru-vllm-server) 권장 사양: Ampere+ 아키텍처, 24GB+ VRAM, 시스템 메모리 32GB+ [근거, WebSearch 결과]
- vlm 추론 프레임워크는 sglang에서 vllm으로 전환됨 [근거, WebSearch 결과, MinerU changelog]

## 대상 환경 판단

- [확정] 유저 서버 GPU: RTX 3080 (Ampere)
- 3080(10GB)에서 pipeline은 여유 충분, vlm은 최소치 충족하나 여유 약 2GB로 빠듯.
- [추측] 기본값 pipeline, 품질 미달 시 vlm 시험하는 순서가 안전.
