# Grok 4.5 가격 조사 (2026-07-09)

## 확인된 사항

- Grok 4.5는 2026-07-08 출시 [근거, https://openrouter.ai/x-ai/grok-4.5]
- API 가격: 입력 $2 / 1M tokens, 출력 $6 / 1M tokens [근거, https://openrouter.ai/x-ai/grok-4.5]
  - 유저가 들은 "출력 100만당 6달러" 소문과 일치
- 캐시된 입력: $0.50 / 1M tokens [근거, WebSearch 결과 요약]
- 컨텍스트 윈도우: 500K tokens [근거, https://openrouter.ai/x-ai/grok-4.5]
- OpenRouter 모델 ID: `x-ai/grok-4.5` (xAI 직접 API 모델 ID는 별도 확인 필요, 관례상 `grok-4.5` 추정)
- xAI API는 OpenAI 호환 형식 (openai SDK로 base_url 변경하여 사용 가능)
- 서버측 도구(web search 등)는 별도 과금 ($5 / 1k calls 등) - 본 프로젝트에서는 미사용 예정

- (해소, 2026-07-10) xAI 직접 API 모델 ID는 `grok-4.5`로 실호출 확인 완료. models.list()에서 존재 확인, 번역/요약 정상 동작.
